# 🍕 Seu Maze Pizzas

> **"A gente capricha mais!"** (We go the extra mile!) — Landing page for Seu Maze Pizzas, a local pizzeria in Jaraguá do Sul, SC, Brazil.

---

## ✨ About

Institutional website for **Seu Maze Pizzas**, built to showcase the business, display products, enable WhatsApp ordering, and provide location & opening hours information.

The project focuses on a premium visual experience with modern design, smooth animations, and full mobile responsiveness.

---

## 🚀 Tech Stack

| Technology | Purpose |
|---|---|
| **Node.js + Express** | Backend server |
| **EJS** | Template engine |
| **Bootstrap 5** | Grid system & responsiveness |
| **Font Awesome 6** | Icons |
| **Google Fonts** | Typography (Inter & Outfit) |
| **CSS3** | Custom styling, animations & glassmorphism |
| **Sharp** | Image optimization |

---

## 🎨 Features & Design

- 🖼️ **Hero Section** with auto-sliding background and SVG wave divider
- 🍕 **Product Cards** with hover animations (Pizzas, Burgers, Cucas)
- 📜 **Infinite marquee** with seamless image loop
- 📍 **Embedded Google Map** with address and business hours
- 📱 **Fully responsive** — optimized for mobile and desktop
- 💬 **WhatsApp CTAs** with pulse animation for conversion
- 🎭 **Glassmorphism** on navbar and hero content
- ✨ **Scroll animations** powered by Intersection Observer
- 🌊 **SVG wave dividers** for smooth section transitions

## ⚙️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18+

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/seuMaze.git
cd seuMaze

# Install dependencies
npm install

# Start the server
node app.js
```

The site will be available at `http://localhost:3596`

### Development Mode

```bash
# With nodemon (hot reload)
npx nodemon app.js
```
